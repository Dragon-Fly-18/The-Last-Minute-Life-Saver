Bro 😎🔥 Google Cloud **Starter Tier la publish panna** basically **Cloud Run deploy** pannuvom. Un FastAPI backend ku steps:

## Step 1 — Google Cloud setup

Open:

[Google Cloud Console](https://console.cloud.google.com/?utm_source=chatgpt.com)

1. Create / select project
2. Billing enable pannu (Starter credits/free quota irundhaalum usually billing attach kekkum)
3. Enable:

```text
Cloud Run API
Cloud Build API
Artifact Registry API
```

---

## Step 2 — Backend ready pannu

Backend folder la:

```text
backend/
│
├── main.py
├── calendar_service.py
├── requirements.txt
├── .env
└── Dockerfile   👈 create
```

---

## Step 3 — requirements.txt

Already illa na:

```bash
pip freeze > requirements.txt
```

---

## Step 4 — Dockerfile create pannu

`backend/Dockerfile`

```dockerfile
FROM python:3.11

WORKDIR /app

COPY requirements.txt .

RUN pip install -r requirements.txt

COPY . .

CMD uvicorn main:app --host 0.0.0.0 --port 8080
```

---

## Step 5 — Google Cloud CLI install

Install:

[Google Cloud CLI](https://cloud.google.com/sdk/docs/install?utm_source=chatgpt.com)

Login:

```bash
gcloud auth login
```

Select project:

```bash
gcloud config set project YOUR_PROJECT_ID
```

---

## Step 6 — Deploy

Backend folder ku po:

```bash
cd backend
```

Run:

```bash
gcloud run deploy
```

Questions varum:

Service name:

```text
ai-productivity-backend
```

Region:

select nearby

Allow unauthenticated:

```text
Yes
```

---

## Step 7 — Environment variables add pannu

Cloud Run console:

```text
Cloud Run
 ↓
Your service
 ↓
Edit & Deploy
 ↓
Variables & Secrets
```

Add:

```env
GEMINI_API_KEY=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

---

## Step 8 — URL kidaikum

Example:

```text
https://ai-productivity-backend-xxxxx.run.app
```

Adha frontend `.env` la replace pannu:

Before:

```env
VITE_API_URL=http://127.0.0.1:8000
```

After:

```env
VITE_API_URL=https://ai-productivity-backend-xxxxx.run.app
```

---

## Step 9 — Frontend deploy

Firebase Hosting use panna:

```bash
npm run build
firebase deploy
```

---

Bro un project ku order:

1. Backend → Cloud Run ✅
2. Frontend → Firebase Hosting ✅
3. Test Gemini + Calendar ✅

Idhu dha Google ecosystem deployment 🔥😎
